# DefraAntyInstaling
Program polegający na fizycznym wpisywaniu liter z stworzonego cfg.

Aplikacja do automatycznego wpisywania odpowiedzi.

## Instalacja

```bash
npm install
cd typer
build
cd ..
```

## Uruchomienie

```bash
npm start
```

## Kompilacja do EXE

```bash
npm run build
```

Plik .exe będzie w folderze `dist/`
